# Artificial Intelligence Game Agent

An AI-powered Tic-Tac-Toe application developed for an Artificial Intelligence course. The project demonstrates adversarial search techniques by implementing Minimax, Negamax, and Alpha-Beta Pruning algorithms for intelligent move selection.

## Features

- Human vs AI gameplay
- Variable board sizes
- Minimax search
- Negamax optimization
- Alpha-Beta Pruning
- Heuristic board evaluation
- Interactive graphical interface using Pygame

## Technologies

- Python
- Pygame

## Algorithms

- Minimax
- Negamax
- Alpha-Beta Pruning
- Heuristic Evaluation Functions
